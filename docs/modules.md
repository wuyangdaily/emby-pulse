# 功能模块说明（含使用方法与配置）

本页对 EmbyPulse 核心模块进行“怎么用 + 需要配置什么”的详细说明，适合新手与管理员快速上手。

> ✅ 提示：所有"系统设置"字段都对应系统设置页面与通知与机器人页面中的真实表单项，已按实际 UI 命名整理。

## 0. 端口与访问说明

EmbyPulse 启动后会同时运行两个独立端口：

| 端口 | 用途 | 访问地址 | 需要登录 |
| --- | --- | --- | --- |
| **10307** | 管理员后台（仪表盘、设置、用户管理等） | `http://你的IP:10307` | 是（Emby 管理员账号） |
| **10308** | 用户中心（求片、个人中心） | `http://你的IP:10308` | 是（Emby 普通用户账号） |

> ⚠️ 10308 端口是物理隔离的独立引擎，只允许访问求片页面、登录页和静态资源，无法越权进入管理后台。

**如何把用户中心链接分享给用户**
1. 确认你的服务器公网 IP 或域名
2. 用户中心访问地址为：`http://你的域名或IP:10308`
3. 将此链接发给用户即可，用户使用自己的 Emby 账号密码登录
4. 如果你做了反向代理，可以把 10308 端口映射到自定义域名（如 `https://request.example.com`）

## 1. 全景仪表盘（Dashboard）

**功能说明**
- 展示当前并发播放、客户端类型、转码负荷
- 展示累计播放次数、播放时长与活跃用户
- 支持按天 / 周 / 月的趋势图

**如何访问**
- 管理员后台：`http://你的IP:10307` → 登录后默认进入仪表盘首页

**如何使用**
1. 登录后自动进入仪表盘
2. 顶部可查看实时播放会话（客户端、IP、转码状态）
3. 中部展示资源总量与播放统计
4. 底部趋势图可切换"天 / 周 / 月"维度
5. 点击用户排行榜可查看更多详情

**设置路径（系统设置页面）**
- 媒体服务器核心：`emby_host` / `emby_api_key` / `playback_data_mode`
- 隐藏黑名单：`hidden_users`

## 2. 缺集管理（Gap Management）

**功能说明**
- 扫描 Emby 剧集与 TMDB 数据，找出缺失集
- 可联动 MoviePilot 搜索资源并补全
- 可截胡下载器，仅下载缺失集

**如何使用**
1. 进入“缺集管理”页面
2. 点击“扫描缺集”并等待结果
3. 选择缺集条目进行“搜索 / 下发 / 截胡”操作

**设置路径（分两处）**
- 系统设置 → MoviePilot 自动化联控：`moviepilot_url` / `moviepilot_token` / `pulse_url`
- 缺集管理页面 → “底层接管配置”弹窗：`client_type` / `client_url` / `client_user` / `client_pass`

## 3. 追剧日历（TV Calendar）

**功能说明**
- 自动展示本周更新剧集
- 已入库可直达 Emby 播放
- 未入库可复制搜索指令

**如何使用**
1. 打开“追剧日历”页面
2. 点击日期查看更新内容
3. 绿色为已入库，红色为缺失

**设置路径（系统设置页面）**
- 无需额外配置（依赖 `emby_api_key` 与 TMDB 数据）

## 4. 求片系统（Request Center）

**功能说明**
- 用户提交电影 / 剧集 / 季度请求
- 管理员审核并流转状态
- 入库后自动闭环

**如何访问**
- 用户端：通过 `http://你的IP:10308` 访问（独立端口，物理隔离）
- 管理端：通过 `http://你的IP:10307` 后台左侧菜单进入"求片管理"

**如何使用（用户）**
1. 打开 `http://你的IP:10308`，使用 Emby 账号密码登录
2. 在"全站大厅"搜索想看的影片
3. 点击提交求片请求
4. 在"个人中心"查看请求状态

**如何使用（管理员）**
1. 登录管理后台 `http://你的IP:10307`
2. 进入"求片管理"页面
3. 审核、下发或拒绝请求
4. 入库完成后系统自动更新状态并通知用户

**设置路径（系统设置 / 通知与机器人页面）**
- 对外服务门户：`emby_public_url` / `client_download_url` / `welcome_message`
- Telegram 通道：`tg_bot_token` / `tg_chat_id`

## 5. 用户管理（User Center）

**功能说明**
- 查看用户资料、到期时间、备注
- 续期、禁用、重置等管理操作
- 用户活跃度与观影趋势
- 批量续期、批量套用权限模板

**如何访问**
- 管理员后台：`http://你的IP:10307` → 左侧菜单"用户管理"

**如何使用**
1. 管理员登录后台，点击左侧"用户管理"
2. 点击某个用户的"配置"按钮查看详情
3. 在弹窗中可切换"基础资料 / 库权限 / 高级策略"三个 Tab
4. 修改到期时间、备注、权限后点击"保存配置"
5. 勾选多个用户可使用底部"批量续期 / 批量套用"操作栏

**邀请系统使用**
1. 点击右上角"生成邀请"按钮
2. 选择有效期（7 天 / 30 天 / 永久）
3. 复制生成的邀请链接发给新用户
4. 新用户打开链接即可自助注册（无需管理员干预）
5. 点击"邀请管理"可查看所有邀请码的使用状态

**设置路径（系统设置页面）**
- 媒体服务器核心：`emby_host` / `emby_api_key`

## 9. Telegram 机器人

**功能说明**
- 播放开始 / 停止推送
- 入库通知与海报发送
- 命令交互（/search /stats /check）
- 黑名单客户端拦截告警

**如何配置**
1. 在 Telegram 中找到 `@BotFather`，发送 `/newbot` 创建机器人
2. 获取 Bot Token
3. 获取你的 Chat ID（可通过 `@userinfobot` 获取）
4. 登录管理后台 → 通知与机器人页面
5. 在"推送场景总控"中开启"启用底层引擎"
6. 在"Telegram 通道"中填入 `Bot Token` 和 `Admin Chat ID`
7. 点击"发送测试"验证连通性
8. 点击"保存配置"

**可用命令**
- `/search 关键词` — 搜索 Emby 媒体库
- `/stats` — 查看服务器统计数据
- `/check` — 系统健康诊断

**设置路径（通知与机器人页面）**
- 推送场景总控：`enable_bot` / `enable_notify` / `enable_library_notify`
- Telegram 通道：`tg_bot_token` / `tg_chat_id`
- 企业微信(推送)：`wecom_corpid` / `wecom_agentid` / `wecom_corpsecret` / `wecom_touser`
- 企微交互指令回调：`wecom_token` / `wecom_aeskey`

## 11. 播放历史（History）

- 查看全站用户的详细观影流水
- 支持按用户筛选、按影视名称搜索
- 展示播放时间、时长、客户端等详细信息

**如何访问**
- 管理员后台：`http://你的IP:10307` → 左侧菜单"播放历史"

**如何使用**
1. 进入"播放历史"页面
2. 顶部下拉框可选择"所有用户"或指定用户
3. 搜索框可按影视名称过滤
4. 列表展示每条播放记录的详细信息

## 12. 内容风云榜（Content Ranking）

**功能说明**
- 全服播放热度与时长综合排行
- 支持按"电影 / 剧集"分类
- 支持按"播放量 / 总时长"排序
- 支持按用户维度查看个人榜单

**如何访问**
- 管理员后台：`http://你的IP:10307` → 左侧菜单"风云榜"

**如何使用**
1. 进入"风云榜"页面
2. 顶部切换"全局 / 电影 / 剧集"分类
3. 切换"播放量 / 总时长"排序维度
4. 下拉框可选择"全服"或指定用户

## 13. 终端态势与管控（Clients）

**功能说明**
- 分析全服设备类型与客户端分布
- 一键阻断异常客户端
- 软件黑名单配置

**如何访问**
- 管理员后台：`http://你的IP:10307` → 左侧菜单"终端管控"

**如何使用**
1. 进入"终端管控"页面
2. 查看设备统计与客户端分布
3. 点击"软件黑名单配置"可添加需要拦截的客户端名称
4. 被拦截的客户端将无法正常播放

> ⚠️ 黑名单拦截依赖 Webhook，请确保已在 Emby 中配置 Webhook 地址。

## 14. 风险管控（Risk Control）

**功能说明**
- 实时雷达监控当前在线账号与并发状态
- 并发超限自动告警
- 历史审计日志查看
- 可配置并发限制规则

**如何访问**
- 管理员后台：`http://你的IP:10307` → 左侧菜单"风控中心"

**如何使用**
1. 进入"风控中心"页面
2. "实时雷达"Tab 展示当前在线账号、并发警报、活动设备数
3. "历史审计"Tab 查看历史违规记录
4. 右上角齿轮按钮可配置并发限制规则

## 15. 去重管理（Dedupe）

**功能说明**
- 智能扫描媒体库中的重复文件（电影 / 剧集）
- 按画质、大小等维度对比重复项
- 支持白名单保护与扫描配置
- 释放无效存储空间

**如何访问**
- 管理员后台：`http://你的IP:10307` → 左侧菜单"去重管理"

**如何使用**
1. 进入"去重管理"页面
2. 点击"智能全库扫描"开始扫描
3. 扫描完成后切换"电影 / 剧集"Tab 查看重复项
4. 对比每组重复文件的画质、大小等信息
5. 选择需要删除的副本进行清理
6. 点击"白名单"可保护不想被扫描的内容
7. 点击"扫描配置"可调整扫描规则
